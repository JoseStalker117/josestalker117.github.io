Proyecto BioInsight Modbus

Este proyecto se complementa con los módulos Atlas del sistema BioInsight. En esta fase, desarrollamos un módulo especializado en medir la calidad del aire de los reactores de microalgas. Esto es clave para el monitoreo de las condiciones ambientales y para optimizar el proceso de crecimiento de las microalgas utilizadas en biotecnología. El módulo se conecta a una plataforma de monitoreo y genera alertas en tiempo real, integrando sensores IoT para recopilar datos precisos.

Año: 2025 Tecnologías: Modbus, IoT, React, Microcontroladores

Inicio del proyecto BioInsight

En este proyecto, definimos y desarrollamos una solución integral para el personal del laboratorio de remediación de la Facultad de Agronomía. La solución permite almacenar, consultar y analizar la información generada por los kits de aquaponics (denominados Módulos Atlas), facilitando la toma de decisiones en tiempo real. La arquitectura de la solución es en la nube, lo que permite acceder a los datos desde cualquier lugar. Además, se desarrolló una aplicación móvil nativa para Android y una página web interactiva usando React, brindando una interfaz de usuario accesible y robusta.

Año: 2024 Tecnologías: React, Android, AWS, Arquitectura en la Nube, IoT

Instalación de kits aquaponics

Este proyecto consistió en el diseño, armado y configuración de sistemas aquapónicos, los cuales integran cultivos vegetales con peces para crear un sistema de producción sostenible. Además, se capacitó a personal técnico sobre el manejo y mantenimiento de los kits, asegurando una correcta operación a largo plazo. Durante este proceso, también se calibraron los sensores y otros dispositivos necesarios para el monitoreo de variables como la temperatura, el pH, y la oxigenación del agua.

Año: 2024 Tecnologías: Aquaponics, Sensores, IoT, Capacitación

Proyecto integrador II

Este proyecto de grado se centró en el desarrollo de una aplicación móvil y web para la gestión de pedidos y menú de cafeterías. La plataforma permite a los usuarios ver el menú, realizar pedidos y recibir notificaciones sobre el estado de su pedido. La integración de una consola IoT es lo que distingue este sistema, pues permite a los empleados de la cafetería ver información en tiempo real sobre los pedidos en curso, usando señales visuales y sonoras para agilizar el despacho de productos. La solución fue diseñada con una arquitectura escalable y responsive para dispositivos móviles y de escritorio.

Año: 2024 Tecnologías: IoT, React, Node.js, App Móvil

Remediación Inyector de CO2

Este proyecto consistió en la creación de un sistema económico y sencillo para controlar el crecimiento de microalgas mediante la inyección controlada de CO2. Utilizando una plataforma Arduino, se diseñó un dispositivo que mide y regula la cantidad de CO2 inyectado en los reactores, optimizando el proceso de fotosíntesis de las microalgas. Este proyecto fue reconocido con el primer lugar en el concurso de carteles CUNORTE en Guadalajara, por su simplicidad, bajo costo y efectividad. El dispositivo se utiliza para mejorar la productividad en bioreactores de microalgas, lo cual tiene aplicaciones en biotecnología y la producción de biocombustibles.

Año: 2025 Tecnologías: Arduino, Sensores de CO2, IoT

Proyecto integrador I

Este proyecto consistió en el diseño y fabricación de un refrigerador portátil utilizando una celda Peltier, que es un dispositivo termoeléctrico. La idea era crear un sistema de refrigeración que no dependiera de compresores o gases refrigerantes, haciendo el dispositivo más eficiente y ecológico. Además, se implementaron técnicas de optimización de recursos, permitiendo una mayor autonomía de funcionamiento sin la necesidad de recurrir a una fuente de energía externa. El refrigerador fue diseñado para usarse en situaciones donde la portabilidad y la eficiencia energética son esenciales, como en actividades al aire libre o en emergencias.

Año: 2020 Tecnologías: Celda Peltier, Electrónica, Energías Renovables
